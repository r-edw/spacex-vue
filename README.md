# spacex-vue

Simple Vue.js application for viewing SpaceX data from [SpaceX API](https://github.com/r-spacex/SpaceX-API)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
